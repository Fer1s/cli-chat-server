# cli-chat server

cli-chat is a simple chat app build with socket.io.

## Install

```bash
$ git clone https://github.com/Ferizzz/cli-chat-server
$ cd cli-chat-server
$ npm install
```

## Start

Development
```bash
$ npm run start
```
Development with nodemon
```bash
$ npm run start:dev
```
Using pm2
```
pm2 start index.js --name cli-chat-server
```
